C, gcc, Makefile

This program was made during the ETNA's "Piscine" camp, during which I had to recreate a lot of native functions of C, like strlen(), strcpy().... only through algorithmic manipulation of pointers with memory management in mind.
my_crd is a simple CLI based CRUD written in C. It takes for input stdin. The implementation makes uses of linked list. This project allowed me to have a better understanding of the C programming language.
