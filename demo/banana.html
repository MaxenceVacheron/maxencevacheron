Banana

Docker, PHP, Symfony, Doctrine, Twig, JavaScript, AJAX, MySQL(MariaDB), HTML5/CSS3

Have you ever put your playlist on shuffle and have a specific song be the exact one you need to listen to at this time ?
Well, this project allows you to create "Moods" or "Tags" and assign them to your songs.
Banana is a music playlist builder with dynamic queue management that is based on restrictive (for now) filters/moods. This allows you to ask for very specific song, for exemple "French, Rock, energy level 2", for a smooth Sunday wake up.
I made Banana in order to learn Doctrine and Twig. If I had to (and I probably will ) restart this project, I wouldn't choose these technologies anymore and would probably go for a PWA client and a node API.
The client makes one AJAX call to an API to generate the playlist. Each song is hosted on the server. If you dig into the code, you will realise that the demo only use a fraction of the available features (and ugly code). You can try it by clicking on the blinking icon (mobile advised).
